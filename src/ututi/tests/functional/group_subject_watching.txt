Group subject watching
======================

    >>> browser = Browser.logIn()

    >>> browser.open('http://localhost/group/moderators')
    >>> browser.getLink('Subjects', url='moderators').click()

    >>> browser.printQuery("id('watched-subjects')")
    <ul id="watched-subjects">
      <li id="empty_subjects_msg" class="empty_msg ">
      Your group is not watching any subjects. Add them by using the search.
    </li>
    </ul>

    >>> browser.printQuery("id('search-results')//a[@class='item-title larger']/text()")
    Matematinė analizė

    >>> browser.getLink('Watch', url='mat_analize').click()
    >>> browser.printQuery("id('watched-subjects')//a/text()")
    Matematinė analizė

    >>> browser.printQuery("id('search-results')")

    >>> browser.open(browser.url)

    >>> browser.getLink(url='unwatch_subject').click()

    >>> browser.printQuery("id('watched-subjects')")
    <ul id="watched-subjects">
      <li id="empty_subjects_msg" class="empty_msg ">
      Your group is not watching any subjects. Add them by using the search.
    </li>
    </ul>

    >>> browser.printQuery("id('search-results')//a[@class='item-title larger']/text()")
    Matematinė analizė
