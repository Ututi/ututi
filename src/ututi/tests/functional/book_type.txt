Tests for book types administration
===============================

The admin should have a new link on his dashboard, that lets him view all book types.

    >>> browser = Browser.logIn()
    >>> browser.open('http://localhost/admin')
    >>> browser.getLink('Book types').click()
    >>> browser.url
    'http://localhost/admin/book_types'

Admin can create book type.

    >>> form = browser.getForm('book_type_form')
    >>> form.getControl('Name').value = 'VadovÄ—lis'
    >>> form.getControl('URL name').value = 'vadoveliai'
    >>> form.getControl('Save').click()

    >>> browser.url
    'http://localhost/admin/book_types'

    >>> browser.printQuery('id("book_types_list")//td/text()')
    Vadov&#279;lis
    vadoveliai


Admin can edit book type.

    >>> browser.getLink('Edit').click()
    >>> form = browser.getForm('book_type_form')
    >>> form.getControl('Name').value = 'Pratybos'
    >>> form.getControl('URL name').value = 'pratybos'
    >>> form.getControl('Edit').click()
    >>> browser.url
    'http://localhost/admin/book_types'
    >>> browser.printQuery('id("book_types_list")//td/text()')
    Pratybos
    pratybos

Admin can delete book type.

    >>> browser.url
    'http://localhost/admin/book_types'
    >>> browser.getLink('Edit').click()
    >>> browser.getLink('Delete').click()
    >>> browser.url
    'http://localhost/admin/book_types'
    >>> browser.printQuery('id("book_types_list")//td/text()')
