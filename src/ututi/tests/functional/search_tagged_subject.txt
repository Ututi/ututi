Searching for tagged subjects
=============================

We log in.

   >>> browser = Browser.logIn()

Tag the subject.

   >>> browser.open('http://localhost/subject/vu/mat_analize/edit')
   >>> browser.getControl('Tags').value = 'matematika'
   >>> browser.getControl('Save').click()

Go to the search form.

   >>> browser.open('http://localhost/profile/search')

Try looking for a subject that does not exist, while passing a tag that exists.

   >>> browser.getControl(name="text").value = 'analizai'
   >>> browser.getControl(name='tags').value = 'matematika'
