Tests for school grades administration
===============================

The admin should have a new link on his dashboard, that lets him view and manage school grades:

    >>> browser = Browser.logIn()
    >>> browser.open('http://localhost/admin')
    >>> browser.getLink('School grades').click()
    >>> browser.url
    'http://localhost/admin/school_grades'

Admin can create new school grade:

    >>> form = browser.getForm('school_grade_form')
    >>> form.getControl('Name').value = '1 kl.'
    >>> form.getControl('Save').click()

    >>> browser.url
    'http://localhost/admin/school_grades'

    >>> browser.printQuery('id("school_grades_list")//td/text()')
    1 kl.


Admin can edit school grades:

    >>> browser.getLink('Edit', url='school').click()
    >>> form = browser.getForm('school_grade_form')
    >>> form.getControl('Name').value = '1 gimnazijos kl.'
    >>> form.getControl('Edit').click()
    >>> browser.url
    'http://localhost/admin/school_grades'
    >>> browser.printQuery('id("school_grades_list")//td/text()')
    1 gimnazijos kl.

Admin can delete school grade:

    >>> browser.url
    'http://localhost/admin/school_grades'
    >>> browser.getLink('Delete').click()
    >>> browser.url
    'http://localhost/admin/school_grades'
    >>> browser.printQuery('id("school_grades_list")//td/text()')
