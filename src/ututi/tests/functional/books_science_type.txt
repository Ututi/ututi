Tests for book science types administration
===========================================

The admin should have a new link on his dashboard, that lets him view all active and inactive group coupons.

    >>> browser = Browser.logIn()
    >>> browser.open('http://localhost/admin')
    >>> browser.getLink('Book science types').click()
    >>> browser.url
    'http://localhost/admin/science_types'

Admin can create science type.

    >>> form = browser.getForm('science_type_form')
    >>> form.getControl('Name').value = 'Humanitariai mokslai'
    >>> form.getControl(name='department').value = ['school']
    >>> form.getControl('Save').click()

    >>> browser.url
    'http://localhost/admin/science_types'

    >>> browser.printQuery('id("science_types_list")//td/text()')
    Humanitariai mokslai
    School books

Admin can edit science type.

    >>> browser.getLink('Edit', 'science').click()
    >>> form = browser.getForm('science_type_form')
    >>> form.getControl('Name').value = 'neHumanitariai mokslai'
    >>> form.getControl(name='department').value = ['university']
    >>> form.getControl('Edit').click()

    >>> browser.url
    'http://localhost/admin/science_types'

    >>> browser.printQuery('id("science_types_list")//td/text()')
    neHumanitariai mokslai
    Students books

Admin can delete city.

    >>> browser.url
    'http://localhost/admin/science_types'
    >>> browser.getLink('Edit', url='science').click()
    >>> browser.getLink('Delete').click()
    >>> browser.url
    'http://localhost/admin/science_types'
    >>> browser.printQuery('id("science_types_list")//td/text()')
