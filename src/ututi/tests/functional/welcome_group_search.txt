Group search in welcome page
============================

    >>> browser = Browser.logIn()

    >>> browser.open('http://localhost/welcome')
    >>> browser.getControl(name='location-0').value = 'Vilniaus Universitetas'
    >>> browser.getControl('entrance year').value = '2003'
    >>> browser.getControl('Search', index=1).click()

After clicking on the search button, we should be in the group search view.

    >>> print browser.url
    http://localhost/findgroup

The link to create a new group should be available at all times.

    >>> browser.getLink('Create your group!').click()

Clicking on it should pass on the entered location values and year to the group creation form.

    >>> print browser.getControl(name='location-0').value
    Vilniaus universitetas

    >>> print browser.getControl(name='location-1').value

    >>> print browser.getControl('Year').value
    ['2003']
