Tests for cities administration
===============================

The admin should have a new link on his dashboard, that lets him view all active and inactive group coupons.

    >>> browser = Browser.logIn()
    >>> browser.open('http://localhost/admin')
    >>> browser.getLink('Cities').click()
    >>> browser.url
    'http://localhost/admin/cities'

Admin can create city.

    >>> form = browser.getForm('city_form')
    >>> form.getControl('Name').value = 'Vilnius'
    >>> form.getControl('Save').click()

    >>> browser.url
    'http://localhost/admin/cities'

    >>> browser.printQuery('id("cities_list")//td/text()')
    Vilnius


Admin can edit city.

    >>> browser.getLink('Edit', url='city').click()
    >>> form = browser.getForm('city_form')
    >>> form.getControl('Name').value = 'Kaunas'
    >>> form.getControl('Edit').click()
    >>> browser.url
    'http://localhost/admin/cities'
    >>> browser.printQuery('id("cities_list")//td/text()')
    Kaunas

Admin can delete city.

    >>> browser.url
    'http://localhost/admin/cities'
    >>> browser.getLink('Edit', url='city').click()
    >>> browser.getLink('Delete').click()
    >>> browser.url
    'http://localhost/admin/cities'
    >>> browser.printQuery('id("cities_list")//td/text()')
